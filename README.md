# Aleksi's Chrome Extensions

This repository contains a collection of custom Google Chrome extensions built for Aleksi.

## Included Extensions

- **Auto Refresh by Aleksi**: Custom timer to automatically refresh pages based on website alias or URL patterns.
- **Auto Fill by Aleksi**: Automatically fill specific text inputs on targeted websites.

## Manual Installation Instructions

Follow these steps to install any of the extensions manually in Google Chrome:

1. **Download the Repository**:
   Download or clone this entire repository to your local computer (e.g., `C:\Users\moxch\Documents\GitHub\aleksi-chrome-extensions`).

2. **Open the Chrome Extensions Page**:
   Open Google Chrome, type `chrome://extensions/` into the address bar, and hit **Enter**.

3. **Enable Developer Mode**:
   In the top right corner of the Extensions page, toggle the **Developer mode** switch so it is turned **ON**.

4. **Load the Extension**:
   - Click the **Load unpacked** button that appears in the top left corner.
   - A file dialog will open. Navigate to the specific extension folder you want to install (for example, `Auto Refresh by Aleksi` or `Auto Fill by Aleksi`).
   - Select that folder and click **Select Folder**.

5. **Pin the Extension (Optional but recommended)**:
   - Click the **Puzzle** icon in the top right of your Chrome browser (next to your profile picture).
   - Find the extension in the list and click the **Pin** icon next to it so it stays visible in your toolbar for easy access.

**Note:** You need to repeat steps 4 and 5 for each individual extension you wish to install from this repository.
