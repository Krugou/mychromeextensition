# Emoji Picker by Aleksi

A quick-access popup menu with categorized emojis and kaomojis.

## Chrome Web Store Listing Details

### Store Listing

**Title:** Emoji Picker by Aleksi

**Short Description:** A quick-access popup menu with categorized emojis and kaomojis.

**Detailed Description:**
> Finding the right emoji shouldn't break your flow. Emoji Picker provides an instant popup menu packed with standard emojis and popular kaomojis. Search by keyword, copy with a single click, and paste it anywhere. Whether you need a simple thumbs up or a classic shrug ¯\_(ツ)_/¯, it's just one click away.

**Category:** Productivity

**Language:** English (United States)

---

### Privacy Practices

**Single Purpose:**
> The single purpose of this extension is to provide a quick popup utility for finding and copying emojis to the clipboard.

**Permission Justification:**
> - `activeTab` / `scripting`: Needed to optionally inject the copied emoji directly into the active text input on the current page in future updates. Currently, it copies to the clipboard.

**Data Usage:**
> This extension does not collect, store, or transmit any user data. It complies fully with Chrome's Limited Use policy.
