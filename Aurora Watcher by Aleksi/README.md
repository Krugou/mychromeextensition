# Aurora Watcher by Aleksi

A simple and intuitive Chrome Extension designed to give you one-click access to the Aurora Watcher monitoring system.

## Chrome Web Store Listing Details

If you are publishing this extension to the Google Chrome Web Store, you can use the following details for your store listing and privacy practices.

---

### Store Listing

**Title:** Aurora Watcher by Aleksi
*(or just "Aurora Watcher" if the name is taken)*

**Short Description:** One-click access to the Aurora Watcher monitoring system.

**Detailed Description:**
> Never miss an aurora borealis display again. The Aurora Watcher extension provides instant, one-click access to the Aurora Watcher monitoring system and station tracking dashbord. By clicking the extension icon, you are immediately taken to the live monitoring platform, allowing you to check space weather, nearby magnetometer stations, and local aurora activity without having to navigate through bookmarks or type URLs.

**Category:** Productivity (or Weather)

**Language:** English (United States)

---

### Privacy Practices

**Single Purpose:**
> The single purpose of this extension is to provide users with a quick, one-click shortcut to open the Aurora Watcher web application (https://krugou.github.io/aurorawatcher/).

**Permission Justification:**
> - *No special permissions are required or requested by this extension.* The extension only utilizes the `action` API to listen for icon clicks and open a new tab. It does not require `storage`, `activeTab`, or any other invasive permissions.

**Data Usage:**
> This extension does not collect, store, or transmit any user data. It does not track browsing history, capture keystrokes, or communicate with external servers beyond opening the specified public URL in a new browser tab. It complies fully with Chrome's Limited Use policy.
