# Quick Currency Converter by Aleksi

A simple extension to convert currencies and units quickly without leaving the current tab.

## Chrome Web Store Listing Details

### Store Listing

**Title:** Quick Currency Converter by Aleksi

**Short Description:** Convert currencies and units quickly without leaving the current tab.

**Detailed Description:**
> The Quick Currency Converter provides an immediate, accessible popup to convert standard currencies (USD, EUR) and units (Miles, Kilometers). Say goodbye to opening a new tab just to figure out how far a kilometer is. Simply click the extension icon, type in your amount, and get the conversion instantly!

**Category:** Productivity

**Language:** English (United States)

---

### Privacy Practices

**Single Purpose:**
> The single purpose of this extension is to provide a quick popup utility for converting currencies and units.

**Permission Justification:**
> - `activeTab`: Used to access the current tab to potentially auto-detect highlighted units in the future. Currently, it just ensures the popup runs securely in the current context.

**Data Usage:**
> This extension does not collect, store, or transmit any user data. It complies fully with Chrome's Limited Use policy.
